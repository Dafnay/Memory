<h1>Memory</h1>
Application created to practice React and Tailwind

<h4>  Open and Run the Project:</h4>
   

Clone your repository

```bash
https://github.com/Dafnay/Memory.git
```
Npm install

```bash
npm install
```
Run 
```bash
npm run dev
```

Play
```bash
https://dafnay-memory.netlify.app/
```
